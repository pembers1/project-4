# Power BI Retail Inventory Health Dashboard

## Objective
Developed an interactive inventory reporting dashboard using a synthetic retail dataset to demonstrate Power BI reporting, dashboard development, KPI monitoring, data visualisation, and inventory analytics capabilities.

## Key Features
* KPI Reporting
* Interactive store and category filtering
* Inventory value analysis by category and location
* Stock health distribution monitoring
* Identification of high-risk overstock products
* Inventory Performance Reporting

## Tools Used
* Power BI
* Power Query
* Microsoft Excel
